# ❤️ Heart Disease Predictor

A machine learning-based web application that predicts the likelihood of heart disease using patient health information such as age, cholesterol level, blood pressure, heart rate, ECG results, and other clinical parameters.

## Features

- Interactive Streamlit interface
- Real-time heart disease prediction
- Machine Learning model integration
- Easy-to-use input form
- Fast and responsive results
- Deployment-ready with requirements.txt

## Technologies Used

- Python 3.14.6 (Cloud environment)
- Pandas
- NumPy
- Scikit-learn (latest compatible version)
- Streamlit

## Installation (Local)

```bash
git clone https://github.com/zk747933-jpg/Heart-Disease-Predictor
cd Heart-Disease-Predictor
pip install -r requirements.txt
streamlit run app.py
